Terms: "unary", "binary", "operand"

Operand --> Operand is value in which operators is applied.
As:- 5*2 - Here, 5 and 2 is operand, Sometimes some people call these "argument". Multiply (*) is operators.

Unary - single operand(argument) is call unary.
As:- -5, is a unary operand.

Binary - Two operand(argument) is call binary.
As:- 5*2, 3-2, 5+9, 6/2 etc.

Math
