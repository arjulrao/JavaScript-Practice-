A variable is a “named storage” for data.

We can create the variable three ways

  • let
  • const
  • var

Variable Naming

There are two limitations on variable names in JavaScript:

  1. The name must contain only letters, digits, or the symbols $ and _
  2. The first character must not be a digit.

Note: When the name contains multiple words, camelCase is commonly used.

Note: Variables named apple and APPLE are two different variables.

Note: There is a list of reserved words, which cannot be used as variable names because they are used by the language itself.
