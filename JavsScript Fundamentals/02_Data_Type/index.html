Data Types

A value in JavaScript is always of a certain type.

JS is a "Dynamically Typed" programming language.

There is eight basic data types in JS.

Seven is Primitive and One is Non-Primitive/ Reference Type

    Primitive Data Type
  1. String
  2. Number

    3. There are many operations for numbers, e.g. multiplication *, division /, addition +, subtraction -, and so on.

    Besides regular numbers, there are so-called “special numeric values” which also belong to this data type: NaN, -Infinity, Infinity.

  3. BigInt

    4. BigInt is also one type of number. But JavaScript cannot safely represent integer value larger then (2 53-1)(that’s 9007199254740991), or less then -(2 53-1)(that’s -9007199254740991).

    So we use BigInt for that type of value.

  4. Boolean - True / False
  5. Null- It’s just a special value which represents “nothing”, “empty” or “value unknown”.
  6. Undefined - The meaning of undefined is “value is not assigned”.

    7. Normally, one uses null to assign an “empty” or “unknown” value to a variable, while undefined is reserved as a default initial value for unassigned things.

  7. symbol - is used to create unique identifiers for objects
